## Tech stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Google Fonts** — Syne + DM Mono

## Lokálny vývoj

```bash
npm install
npm run dev
```

## Deploy na Vercel

```bash
# 1. Push na GitHub
git init
git add .
git commit -m "init: AI nástroje 2026"
git remote add origin https://github.com/TVOJE_USERNAME/ai-nastroje-2026.git
git push -u origin main

# 2. Import projektu na vercel.com → New Project → GitHub repo
# Vercel automaticky detekuje Next.js a nasadí.
```
